[ pyKali # ReadME ]

Run Setup.py Using "python setup.py" command.
Choose Your Username and Password to config the root Directory.
Install Required Libraries.
Done :)

Now You have a Simulated Kali Linux on your Windows :)

------------- Commands -------------

[ START ] to run the tool , run pykali.py file in the cmd using "python pykali.py" command.
[ STOP ] to stop running and exit tool , use "power --0" command.
[ More ] use "pykali --help" commad to see more information about using tool.

[ Important ] --> Run this tool , JUST IN COMMAND PROMPT (cmd)
